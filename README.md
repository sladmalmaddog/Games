# Python Games

This folder contains my tries at making games in Python. They do not share a menu or launcher; each game has its own folder and its own `main.py`.

All games use Python's standard `tkinter` GUI library, so there are no external pip packages to install on Windows.

## Run a Game

Open a terminal in a game folder and run:

```powershell
python main.py
```

Example:

```powershell
cd Games\Snake
python main.py
```
